📋 MP4変換から登録までの手順
1

「候補抽選」タブでプロンプトを取得

プロンプトをコピーしてGemini/Lyria 3で楽曲を生成してください。

2

採用MP4を格納

生成した楽曲のMP4ファイルを以下に配置してください（タイトルを末尾に付与）：

assets_src/audio/vocalmp4/song_xxx_xxx_xxx<タイトル>.mp4
3

音声変換 + ジャケット + 歌詞生成

ターミナルで実行（台帳に自動登録）：

npm run vocalmp4:convert -- "assets_src/audio/vocalmp4/song_xxx_xxx_xxx<タイトル>.mp4" --register
4

ノーツ生成

変換後のMP3からリズムゲーム用ノーツを自動生成：

node scripts/generate-notes.js public/assets/audio/vocalmp4/song_xxx_xxx_xxx.mp3
5

再生確認

VocalMP4テスターで音源・ジャケット・歌詞を確認：

▶ Player Tester を開く
⚡ コマンド生成

候補IDからコマンドを自動生成します。タイトルは手入力してください。

IDとタイトルを入力してください
