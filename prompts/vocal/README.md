# P0008 Bard in Maghribal 144 subtheme prompts v2 simple

## 内容
- v2シンプルプロンプト版CSVの `gemini_prompt` を1曲1txtに展開。
- ファイル名は `ID_楽器_曲調_主題.txt`。
- 例: `song_101_211_321_フレームドラム_ワルツ_星図観測.txt`
- 各txtの冒頭は `ID: song_...`。
- `prompt_manifest.csv` に一覧を収録。

## プロンプト方針
- 用途: 音ゲー用BGM。無音厳禁。
- 時間: 65〜75秒。
- 華奢で可憐な若い女性ボーカル。
- Geminiにプロンプト整備だけで止まらず楽曲生成まで実行させる。
