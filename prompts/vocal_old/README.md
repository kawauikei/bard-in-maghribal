# Bard in Maghribal vocal prompts

## 配置先
`public/prompts/vocal/`

## ID形式
既存5桁ID + 地域コード1桁。

- `0`: 地域なし。既存テーマ + 人物/事件主題。
- `1`〜`9`, `A`, `B`, `C`: 12地域。既存テーマ + 地域場面。

## 件数
- theme: 72
- region: 72
- total: 144

## 生成方針
- Gemini投入用プロンプト本文に、勇猛/優美/神秘などのゲーム内部属性名は出さない。
- ChatGPT側で内部属性を登場人物、事件、詩情へ翻訳している。
- 歌詞はChatGPTが作詞し、Geminiには厳格遵守を指示する。
- 発音の聞き取りやすさを優先する。
- 同じ短い言葉や同じサビの繰り返しだけで尺を埋めない。
- 通常名詞に不自然な「お」を付ける歌詞を避ける。
- 約60秒、女性ボーカル、主体楽器明瞭、無音余白なし、拍点明瞭。
